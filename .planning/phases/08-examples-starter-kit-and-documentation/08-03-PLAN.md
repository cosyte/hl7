  - **Warnings carry stable codes + positional context** — programmatically react to deviations by `code` string, with `segmentIndex`/`fieldIndex`/etc.
  - **Actively used in production** — dogfooded internally on healthcare-integration projects.

**Section 4 — HL7 in 90 seconds (DOC-04, D-16 ASCII tree)**

- H2: `## HL7 in 90 seconds`
- Two short paragraphs (<= 2 per requirement).
- ASCII tree fenced code block (CONTEXT specifics lines 510-521 — exact tree):
  ```
  Message
   ├── MSH    (header — sender, receiver, type, timestamp)
   ├── EVN    (event details for ADT)
   ├── PID    (patient identification — name, MRN, DOB)
   ├── PV1    (visit — class, location, attending)
   └── OBX×N  (observations — repeats for labs, vitals)

   Each segment   = pipe-delimited (|) fields
   Each field     = caret-delimited (^) components
   Each component = ampersand-delimited (&) subcomponents
  ```
- Closing sentence: "That's enough HL7 to use this library productively. Everything else is in the spec, but you won't need it."
